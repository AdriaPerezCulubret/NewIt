[TABLES]

Article --> indexing the articles.

Source --> indexing the sources, relatied to Feed_Subscription.

Category --> indexing the categories, used for the scrolldown menu of search to specify category to be searched. Defined by us. Related to Search_Subscription.

Feed --> indexing feed information (language, link, etc).

Bookmark --> indexing the articles that need to be saved. Protected from automatic deletion of news. If at least one users saves an article, it is save.

Feed_Subsription --> indexing the feeds where the user is subscribed. Useful for inner data, and relevant to user addition of new sources.

Search_Subscription --> search manager.

Auth_user --> indexing user information.
